(async) login()

Logs in the user by calling the apiService's login method. If the server returns a token, it stores the token in local storage, changes the user and the logged state in the Vuex store. If the server returns a 401 status, it means the credentials are invalid, so it removes the token from local storage and sets the error message. If the server returns other errors, it appends them to the error message.
